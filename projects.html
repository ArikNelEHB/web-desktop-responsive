    Automatic Pet Feeder

    Adjustable pet feeder
    Adjust settings via WiFi
    "work in progress"

    This is a open-source pet feeder that everyone will be able to build
    using 3d printing and basic electronic knowledge.
    (soldering skills required)
    source code and files can be found on my GitHub

    About

    This pet feeder is fully 3d printed, no glueing required.
    It has a food dispenser and a water dispenser, bove can be cleaned
    easily because i have made it easy to disassemble.
    This project runs on a esp32, the user can connect to the generated network
    of the pet feeder and can then change the food/water dispenser interval.
    The pet feeder has a manual mode aswell, interactable via a LCD screen
    and a rotary encoder.

    Adjustable EQ Bluetooth Boombox

    16-bit Bluetooth audio
    Adjustable Audio DSP
    OLED Screen Navigation
    WiFi radio
    "work in progress"

    About

    The boombox it's shell volume has been calculated using WinISP
    thus the dimensions are perfect to achieve the best sound possible.
    It's built using 12mm thick MDF wood.
    It has a built in Audio DSP (used for audio processing) that can be interacted with
    using a web application where the user can select their desired audio profile.
    The boombox can also be configured using the built in buttons and OLED Screen.
    The power output at peak is 120W max.
    The battery pack inside is capable of providing 2 hours of playtime at peak power.
    (Custom 27V litium-ion battery pack with protection features.)
